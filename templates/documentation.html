Anime Info API Documentation

Endpoint: /json/info

GET /json/info

Returns information about a specific anime show in JSON format.

Parameters:

  • anime_id (integer) - The ID of the anime show to retrieve information for.

Response:

  • 200 OK - Returns the anime information in JSON format.
  • 404 Not Found - If the anime with the specified ID is not found.
  • 500 Internal Server Error - If there is an error retrieving the anime information.

Example

Retrieve Anime Information

Example Request:

GET /json/info?anime_id=123

Example Response:

{
    "title": "Attack on Titan",
    "synopsis": "Several hundred years ago, humans were nearly exterminated by Titans...",
    "poster": "https://example.com/poster.jpg",
    "thumbnail": "https://example.com/thumbnail.jpg",
    "english_name": "Attack on Titan",
    "japanese_name": "進撃の巨人",
    "studio": "Wit Studio",
    "genre": "Action, Drama, Fantasy",
    "episodes": 25,
    "episode_duration": 24,
    "start_date": "2013-04-07",
    "end_date": "2013-09-29",
    "score": 90
}

Usage

To use the Anime Info API, make a GET request to the /json/info endpoint with the anime_id parameter to retrieve information about a specific anime show.

Notes

- The anime information is retrieved from AniList API.

- The IDs of the anime shows can be obtained from AniList or any other anime database.
